对使用markdown与html的感想

讲述一些使用它们的感想

返回上一页 返回主页

markdown

这个东西我最大的感想就是：方便快捷！

  • 使用起来非常方便，只需要一两个符号就能表示语义，也就是多少级标题，引用，什么的
  • 学习成本非常低，几分钟就能学会
  • 可以专注于内容的编写，而不是格式的调整
  • 可以很容易地转换成其他格式，比如HTML，PDF，Word等

HTML

这个东西我最大的感想就是：太灵活了！

  • 可以实现各种复杂的布局和效果，比如表格，表单，动画等
  • 可以精细到每个字在css上进行调整
  • 可以通过CSS和JavaScript来增强页面的样式和交互性
  • 可以控制页面的结构和语义，比如使用语义化标签来提高可访问性和SEO
  • 进行控制花费时间比md长一些
  • 学习成本高，大概几天才能把全部内容学完

不管是啥，你得写内容，而写内容才是难度最高的

另外，二者之间可以互相转换，还挺不错的
