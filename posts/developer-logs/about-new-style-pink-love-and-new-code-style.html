新主题的设计

首先便是先设置背景图片，然后选择红心图片设为最开始的图片，接下来设置线条(linear-gradient)，使其形成网格状， 最后设置背景的重复方式、大小、位置以及固定方式， 其中红心因为太大，需要通过其他软件把红心图片的大小缩小到合适的尺寸才行，否则会导致背景图案过大。 这是原图与修改后的图片：

接下来是达到要求的css代码 

css
    html {
        background-color: #ffe4ef;
        background-image: url('../images/style-images/redheart.png'), 
            linear-gradient(to right, #ffd6e0 5px, transparent 5px),
            linear-gradient(to bottom, #ffd6e0 5px, transparent 5px), 
            linear-gradient(to right, #ffd6e0 5px, transparent 5px),
            linear-gradient(to bottom, #ffd6e0 5px, transparent 5px);
        background-repeat: repeat;
        background-size: 320px 320px,320px 320px, 320px 320px, 320px 320px, 320px 320px, 320px 320px;
        background-position: 0 0,155px 155px, 155px 155px, 165px 165px, 165px 165px;
        background-attachment: fixed;
        text-align: center;
    }

以及代码块主题的设计，主要是通过设置代码块的背景颜色为深色、边框、pre属性内的字体颜色等属性来实现的，另外也在pre和code部分用了分割的线条。 当然了，我有时也可以直接插入图片也行，如果要是为了更好突出一些变量之类的东西的话，以后也可能用js或ts之类的来实现。 

css
    pre.code {
    white-space: pre-wrap;
    font-family: 'Fira Code', monospace;
    text-align: justify;
    font-size: 1rem;
    line-height: 1.5;
    color: #d4d4d4;
    padding: 20px;
    border-radius: 15px;
    background-color: rgba(50, 50, 50, 0.9);
    border: 2px solid plum;
    margin-top: 30px;
    margin-bottom: 50px;
    }

    code::before {
    content: "";
    display: block;
    width: 100%;
    height: 2px;
    margin: 10px 0 -10px 0;
    background: linear-gradient(to right, plum, #d4d4d4, plum);
    border-radius: 1px;
    }

    pre.code > :not(code) {
    font-size: 1.3rem;
    font-weight: bold;
    }
